URL del Web App (Google Apps Script)
Pega aquí la URL después de desplegar el script como Web App.
Esta URL conecta la app con Google Sheets y sincroniza todos los dispositivos.
Pasos para configurar Google Sheets:
1. Abre el Google Sheet de referencia
2. Extensiones → Apps Script
3. Pega el código del archivo Code.gs
4. Despliegar → Nueva implementación → Aplicación web
   · Ejecutar como: Yo
   · Acceso: Cualquier usuario
5. Copia la URL y pégala arriba
6. Autoriza el acceso cuando Google lo pida
